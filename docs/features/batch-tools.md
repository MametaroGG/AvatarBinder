# 強力なバッチツールと一括操作

AvatarBinder は、単なるライブラリではなく、日々の制作作業を加速させるためのツール群を搭載しています。

![ツールボックス画面](/screenshot-placeholder.svg)

## 一括操作 (Batch Tools)

AvatarBinder を通じて、以下のような作業を複数アバターに対して一括で実行できます（拡張機能として提供）。
- 修復や重複チェックの一括実行
- 未使用パッケージのクリーンアップ

## 参照の修復 (Repair)

プロジェクト内のファイル移動やリネームによって、ライブラリ内のアバターへのリンクが切れてしまった（Missing になった）場合、このツールが役立ちます。
プロジェクト全体をスキャンし、一致する Prefab を見つけ出して参照を自動的に修復します。

## 重複チェック (Dedupe)

同じアバターが別々の場所に重複して登録されていないか、同一の GUID を持つ Prefab を自動的に検出し、整理を促します。

## シーン同期・サムネ再撮影 (Scene Sync)

シーン上でアバターの衣装を着せ替えたり、アクセサリの表示状態（可視/不可視）を変更した状態を同期し、新しいサムネイルとして再撮影するための機能です。
「シーン可視→プリセット保存」で現在の見た目をツール側に記録し、「プリセットでサムネ再撮影」を押すことで、ワンクリックでライブラリのプレビュー画像を最新の見た目に更新できます。

## パッケージの一掃 (Purge)

インポートした後に不要になった `.unitypackage` ファイルなどを、ライブラリの登録情報に基づいて一括で削除し、ディスク容量を節約します。
