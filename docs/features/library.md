# ライブラリ管理

AvatarBinder のメイン画面では、登録されたアバターを直感的に管理できます。

![メイン画面（ライブラリ）](/screenshot-placeholder.svg)

## 2つの表示モード

- **カードビュー**: サムネイルを大きく表示するモードです。一目でアバターを識別するのに適しています。下部のスライダーでカードのサイズや余白（Gap）を自由に変更できます。
- **リストビュー**: 名前やカテゴリ、メモ、更新日時などを詳細に表示するモードです。大量のアバターを効率的にスクロールして探す際に便利です。

## 強力なフィルタリング

上部のツールバーを使って、必要なアバターを素早く絞り込みます。

- **検索バー**: 名前だけでなく、メモの内容やタグでも部分一致検索が可能です。
- **カテゴリ**: 「男性アバター」「女性アバター」「デフォルメアバター」などのカテゴリで絞り込めます。
- **お気に入り (★)**: 頻繁に使用するアバターにお気に入りマークを付け、それだけを表示することができます。

## データの並び替え

以下の基準でアバターをソートできます。
- 追加日時（新しい順）
- 表示名（A↔Z）
- 最近のインポート順
- よく使う順

## 詳細パネル (右ペイン)

アバターを選択すると、右側に詳細情報が表示されます。
- **プレビュー**: 拡大されたサムネイルと基本情報。
- **タグ管理**: 複数のタグを付与して整理。
- **メモ**: 制作時の注意点や衣装のパーツ構成などを自由にメモできます。
- **パッケージ/操作タブ**: 自動出力（Auto Export）や依存関係の設定、ワンクリックでの `UnityPackage` 出力、および保存先の変更が行えます。各設定はPreferencesと連動しており、「このプロジェクトにインポート」という便利なアクションも可能です。
- **その他のアクション**: 「シーンに配置」「Prefabを開く」「ファイルをエクスプローラーで表示」などの基本操作が行えます。
