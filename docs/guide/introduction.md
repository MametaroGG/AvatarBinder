# AvatarBinderとは

**AvatarBinder** は、VRChat向けアバターの制作・管理を効率化するために開発されたUnityエディタ拡張ツールです。

## 開発の背景

アバター改変を楽しむVRChatユーザーにとって、アバターデータの管理は非常に手間のかかる作業です。
- 「あのアバター、どのフォルダに置いたっけ？」
- 「この衣装を適用したPrefab、どれが最新？」
- 「導入して不要になったパッケージや、重複したPrefabを整理したい」

これらの課題を解決するために、プロジェクト内のアバターやPrefabを一つの「ライブラリ」として統合管理し、直感的なUIで操作できるようにしたのが AvatarBinder です。

## 主な特徴

- **プロジェクトを跨いだ共有**: ライブラリデータはプロジェクト間で共通化されるため、別のUnityプロジェクトからでも登録したアバターを一目で確認可能です。
- **ビジュアルライブラリ**: サムネイル付きのカード形式、または詳細なリスト形式でアバターを一覧表示。
- **瞬時の検索とフィルタ**: 名前、タグ、メモによる強力な検索機能。カテゴリ分けやお気に入り登録にも対応。
- **シームレスな登録と上書き**: Prefabからの登録はもちろん、作業中のアバターの**上書き保存**にも対応。データの増殖を防ぎます。
- **実用的なユーティリティ**: 壊れた参照の自動修復、重複アセットの検出、シーン間での同期確認など、かゆいところに手が届く機能を搭載。
- **Unityエディタ完結**: 外部ツールを介さず、開発環境の中でそのまま管理・操作が可能。
