# インストール方法

AvatarBinder は、UnityPackage 形式で提供されています。手動でインポートしてご利用ください。

## 推奨環境

- **Unity**: 2022.3.6f1 以降 (VRChat推奨バージョン)
- **VRChat SDK**: VRCSDK3-Avatars (最新版を推奨)

## 手動インストール (.unitypackage)

1. [BOOTHの販売ページ](https://mametarovv.booth.pm/items/7336016) にアクセスし、提供されている最新の `AvatarBinder.unitypackage` をダウンロードします。
2. Unityプロジェクトを開いた状態で、ダウンロードしたファイルをダブルクリック、または `Assets > Import Package > Custom Package...` からインポートします。
3. インポート画面で「Import」をクリックします。

---

インストールが完了したら、[クイックスタート](/guide/quick-start)へ進みましょう。
