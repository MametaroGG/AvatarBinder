# クイックスタート

インストールが完了したら、さっそく AvatarBinder を使ってみましょう。

## 1. ウィンドウを開く

Unity上部のメニューから `Tools > Mame2an > Avatar Binder` を選択してウィンドウを開きます。

## 2. アバターを登録する

アバターをライブラリに追加する方法は3つあります。

- **ドラッグ＆ドロップ**: ProjectウィンドウにあるアバターのPrefabを、ウィンドウ上部の「ここにドラッグ＆ドロップ」エリアに放り込みます。
- **シーンから追加**: Hierarchyで選択中のオブジェクトを、下部の「選択中を追加（Scene）」ボタンで登録します。
- **フォルダスキャン**: 「フォルダ追加」ボタンを押し、アバターが入っているフォルダを選択すると、中にあるPrefabを一括で検索・登録します。

## 3. サムネイルを設定する

登録したアバターのカードを選択し、右側のプレビューエリアに画像をドラッグ＆ドロップすることで、好きなサムネイルを設定できます。

## 4. 検索と利用

上部の検索バーに名前やタグを入力してアバターを探します。見つけたアバターをダブルクリック（または右クリックメニュー）することで、シーンへの配置やPrefabの場所の特定が素早く行えます。
